# 运行介绍

该项目是使用PySide6库开发的一个基于YOLOv5的图像识别应用程序。
用户可以通过该应用程序选择图像文件或摄像头实时检测目标物体，并显示检测结果。
同时，该应用程序还支持主动学习功能，用户可以手动标记一些错误的检测结果，以帮助模型进行优化。
不过目前摄像头还没开发好 只能识别图片。
